# Drug Side-Effect Prediction (atopic dermatitis)

Safety-aware extension of the upstream `drug-repurposing/` pipeline. Takes the
ranked drug-repurposing candidates and produces a multi-source safety annotation
plus an ML-based adverse-event probability profile for every candidate.

```
drug-repurposing/                                drug-side-effect/  (this folder)
└── results/                                     │
    └── ranked_candidates_v3_known.csv  ──────►  ├── scripts/v1/   rule-based 6-source annotation
                                                 ├── scripts/v2/   ML AE prediction (12 groups)
                                                 ├── data/         caches (gitignored)
                                                 ├── results/      headline CSVs + per-source intermediates
                                                 └── docs/         literature review + research plan + dev roadmap
```

## Pipeline overview

```
┌────────────────────────────────────────────────────────────────────────┐
│ v1   scripts/v1/                                                       │
│      6-source rule-based annotation:                                   │
│        Open Targets · SIDER 4.1 · OpenFDA FAERS                        │
│        DILI proxy · GWAS-Catalog ADR · repo lit-review                 │
│      derives safety_flag ∈ {RED, AMBER, GREEN, UNKNOWN}                │
│        01_resolve_chembl.py … 08_merge_safety.py                       │
└────────────────────────────────────────────────────────────────────────┘
                               │  results/ranked_candidates_v3_safety.csv
                               │  results/top50_safety_summary_v3.md
                               ▼
┌────────────────────────────────────────────────────────────────────────┐
│ v2   scripts/v2/                                                       │
│      ML multi-label AE prediction across 12 AE groups                  │
│      Morgan-2048 fingerprints + target one-hot + 8-class family        │
│      LR (calibrated) + GradientBoosting per AE group                   │
│      14 new pred_* columns appended to v1 CSV                          │
│        01_build_label_matrix.py … 04_predict_candidates.py             │
└────────────────────────────────────────────────────────────────────────┘
                               │  results/ranked_candidates_v3_safety_v2.csv
                               │  results/v2/model_metrics.csv
                               ▼
                       (planned) v3 — drug-target Mendelian randomization
                       (planned) v4 — integrated benefit-risk tier + Cytoscape
```

See `docs/dev-roadmap.md` for the full versioned plan and `docs/research-plan.md`
for the underlying scientific framework.

## Upstream dependency

This folder reads exactly **one input** from outside its own boundary:

```
drug-repurposing/results/ranked_candidates_v3_known.csv  (1,712 MoA-annotated compounds, ranked)
```

Generated by the sibling `drug-repurposing/` pipeline (steps 1–3: GSE32924
disease signature → BigQuery LINCS dot-product scoring → MoA annotation +
positive-control validation). The path is hardcoded as an absolute reference in
`scripts/v1/01_resolve_chembl.py` and `scripts/v1/08_merge_safety.py` to keep
the cross-folder dependency explicit.
